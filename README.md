# LHUM400

Static tools for LHUM400 — Berklee College of Music.

Currently hosts the **Real-World Readiness Audit**, a self-scoring multiple-choice survey for week one. Students answer each prompt, receive a tailored response per answer, and can print a PDF audit report with QR codes pointing to follow-up resources.

Live: https://kramermusician.github.io/lhum400/

Source HTML lives in the course working folder; this repo is the published copy.
